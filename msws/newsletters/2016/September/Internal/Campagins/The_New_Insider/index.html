'R' LANGUAGE
 
R is a programming language, developed at Bell Laboratories. R is open source, and can run on any operating system and is exclusively used for data analysis, data manipulation, and data handling. R has a package system that allows its users to add their own functionality package which can be downloaded from CRAN (Comprehensive R Archive Network)
