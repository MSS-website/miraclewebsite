Backup Alert = Email Migration
Email Server migration on October 16th, 2015 at 7 PM EST
 
Change is the essence of life. Never in our history before have we seen so many changes happen at Miracle, as we transform ourself for the future. From new team members to more global facilities, everything is changing for the better today at Miracle. Today we've got more new stuff for you.
 
So, What's new now?
We are happy to announce that we will be migrating from our existing WebMail to Zimbra Email communication system which has the following features such as,
  • Advanced Rich Web-Based Email
  • Group scheduling
  • Advanced spam control System
  • Web-Based address Book
  • Web-Based calendar
  • Sharing Folders and Files
  • Web-Based Tasks
  • Personal/Group File Briefcase
  • Unified Communications Integration
  • Outlook Sync/POP/IMAP Email

    When is this happening?
    The IT Team is doing everything they can to execute the migration during a time where minimum work disruption will be caused. Please note that you might have issues with sending out emails during this time period.
    Date : 16th October 2015 (Friday)
    Time : 7:00PM EST
    What should I do before migration?
    As we all know data is too important to loose, so you need to back up all the emails if you are using the Web Client [Using the email in any Browser]. If you are using Email clients like Thunder Bird, Outlook or any other email clients, You do not have to do anything since they are in your local machine.
